# Street Bites Backend API

🍔 RESTful API for Street Bites food delivery system

## Features
- User authentication (JWT)
- Product management
- Stall management
- Order processing
- Image upload (Cloudinary)
- PostgreSQL database

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- Cloudinary
- JWT Authentication

## Installation
```bash
npm install
npm run dev
```

## Environment Variables
Create a `.env` file with:
- DATABASE_URL
- JWT_SECRET
- CLOUDINARY credentials
